The following map highlights the number of respondents to the initial FCC public comment period on its proposed changes for net neutrality rules which have been cited as potentially destroying net neutrality.

The initial comment period ended July 18, 2014. The comment period was later extended to September 15, 2014.

The following data was made available by the FCC in a series of xml files. Some work was required to clean this up into the CSV file below.

Approximately 446,667 items were made available by the FCC. The following represents 435,789 of those items which contained correct zip codes.
